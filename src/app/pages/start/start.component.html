活动介绍

2018年6月23日至6月29日期间，广州IFC国金天地会员可在线参与2018年世界杯竞猜活动，参与活动的会员可通过抽奖获得会员积分，猜中出线国家队伍更可有机会获得XXXX礼品一份。

活动说明

  1. 活动期间，每个微信ID只能参与一次竞猜；
  2. 奖品兑换需提供真实有效的信息，如因为游戏过程中填写错误无效信息导致未能兑换奖品，活动主办方概不承担任何责任；
  3. 如成功竞猜结果，我们将从中抽取5位幸运儿送出礼品，并由专人电话通知中奖者；
  4. 活动规则如有相关变动或调整，将公布在活动页面上即时生效，恕不另行通知；
  5. 活动解释权在法律允许范围内归广州IFC国金天地所有。
